Руководство пользователя SportIduino Lite

 

SportIduino Lite – это простая программа для смартфонов Android с NFC для работы с чипами и станциями системы отметки SportIduino. Основное предназначение - настройка станций КП в лесу. Также можно инициализировать и читать чипы участников, например, на небольшой тренировке.

 

Текущая версия: 1.2.0.

Поддерживаемые чипы: прежде всего, NTag215. Также работают Ntag213, NTag216, MifareClassic1K.

Поддерживаемые станции: SportIduino

 

Функционал:

1.  Настройка станций. Позволяет создать служебный («мастер») чип, который при взаимодействии со станцией установит на ней одно из:

А) ее номер

Б) текущее время

В) усыпит ее (hibernate)

Г) изменит пароль, который используется служебными чипами. Пароль по умолчанию – 0.0.0. Установка пароля на станцию сейчас корректно работает только для станций Александра Воликова по причине разных форматов записи у него и у других владельцев станций.

При этом станция стирает информацию со служебного чипа.

                  

2. Запись чипа. Чип очищается и на него записывается номер участника. Это занимает около 3 секунд. По окончанию звучит один длинный сигнал. Если убрать чип раньше, появится сообщение об ошибке и три коротких звуковых сигнала. При массовой записи чипов удобно включить автоматическое увеличение номера.

        

3. Чтение результатов с чипа участника. Выводится номер участника, количество отметок КП и интервал между крайними отметками. Для каждой отметки отображается номер КП, интервал от первой отметки и время отметки в формате ЧЧ:ММ:СС. Данные никуда не сохраняются. Для служебного чипа выводится информация о нем.

        

4. В меню ⋮ можно задать пароль станций для записи на служебные чипы.

 

Ограничения:

Эта программа намеренно реализует только базовые функции. В ней нет:

А) сохранения данных,
Б) передачи данных,
В) защиты паролем самогó чипа.

Программа распространяется «как есть». Исходный код не опубликован.
